BigchainDB와 Byzantine Fault Tolerance¶

BigchainDB Server 는 블록체인 합의와 트랜잭션 복제에 Tendermint를 사용합니다.

그리고 Tendermint 는 Byzantine Fault Tolerant (BFT).
